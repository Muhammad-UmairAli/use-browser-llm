# Task: WebGPU capability detection utility

## Description

A small, synchronously-callable utility that detects whether the current browser/device can run WebGPU-backed inference, returning a typed result (supported / unsupported + reason) rather than throwing. This backs the hook's `unsupported` state (task 007) and must run before any model load is attempted.

## Acceptance Criteria
- [ ] Exported `detectWebGPUSupport()` returns a discriminated union (`{ supported: true }` | `{ supported: false, reason: ... }`)
- [ ] Detects absence of `navigator.gpu`
- [ ] Detects adapter-request failure (`requestAdapter()` resolving to `null`)
- [ ] Unit tested with `navigator.gpu` mocked present/absent/failing

## Technical Details

Runs on the main thread (capability detection does not need the worker). Pure function, no side effects, safe to call multiple times.

## Dependencies

Depends on task 001 (build tooling) to have a place to add the module and run tests against it.
