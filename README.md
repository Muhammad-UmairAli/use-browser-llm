# use-local-llm

A headless React hook for running Large Language Models locally in the
browser via WebGPU — no backend, no API keys, no data leaving the device.

> **Status:** early scaffolding (Phase 1 in progress). The public
> `useLocalLLM()` API is not implemented yet — see
> `.claude/epics/use-local-llm/epic.md` for the build plan. Full usage docs
> land in task P1-09.

This package is intentionally headless: it ships no UI components and no
styles, only hooks. Bring your own design system.
