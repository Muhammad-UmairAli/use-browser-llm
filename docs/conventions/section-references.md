# Canonical section reference descriptors

Every numbered section reference in this project uses the form `<file-stem> §<number> (<descriptor>)`. Both the file and the descriptor are included **every time**, not just on first mention. Never bare `§N`. Applies to chat, commits, PR/Issue bodies, doc cross-references, and console output from kit tools.

This file is the single source of truth for descriptors. When a new numbered section is added anywhere in the project, add a row here so references stay consistent.
