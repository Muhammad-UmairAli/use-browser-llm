# Architecture & design decisions

Numbered, dated decisions with their rationale and current status. New decisions get the next number; superseded decisions stay (marked superseded) so the history is intact.

## Decision template

```
## Decision N — <one-line title>

- **Date:** YYYY-MM-DD
- **Status:** open / decided / superseded by Decision M
- **Context:** what prompted this decision
- **Decision:** what was chosen
- **Alternatives considered:** what else was on the table and why those were rejected
- **Consequences:** what this enables and what it costs
```
